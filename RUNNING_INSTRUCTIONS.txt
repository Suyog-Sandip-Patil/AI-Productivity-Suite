üõ†Ô∏è TROUBLESHOOTING
================================================================================

COMMON ISSUES AND SOLUTIONS:

1. PORT ALREADY IN USE
   Error: "Port 3000 is already in use"
   Solution: 
   - Kill existing processes: npx kill-port 3000
   - Or use different port: PORT=3001 npm start

2. MODULE NOT FOUND
   Error: "Cannot find module..."
   Solution:
   - Delete node_modules: rm -rf node_modules
   - Reinstall: npm install
   - For client: cd client && rm -rf node_modules && npm install

3. CORS ERRORS
   Error: "CORS policy blocked"
   Solution: Backend is configured for CORS, restart both servers

4. VOICE FEATURES NOT WORKING
   Issue: Speech recognition not responding
   Solution: 
   - Use HTTPS or localhost (required for Web Speech API)
   - Check browser permissions for microphone
   - Supported browsers: Chrome, Edge, Safari

5. DATABASE/STORAGE ISSUES
   Issue: Data not persisting
   Solution: App uses localStorage - check browser settings
